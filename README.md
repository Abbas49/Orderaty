# Orderaty - Multi-Vendor E-Commerce Platform

Orderaty is a comprehensive e-commerce platform built with ASP.NET Core MVC, featuring four distinct user roles (Admin, Client, Seller, Delivery) and complete order management functionality.

## Quick Start

### Prerequisites
- .NET 8.0 SDK
- SQL Server

## Technology Stack

- ASP.NET Core 8.0 MVC
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity

## Features

- Multi-role user system (Admin, Client, Seller, Delivery)
- Product catalog and management
- Shopping cart and checkout
- Order management and tracking
- Review and rating system
- Delivery management
- Admin dashboard with analytics
