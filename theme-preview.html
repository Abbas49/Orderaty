ðŸ’» For Developers

Themes are implemented using CSS custom properties:

/* Access theme colors in your CSS */ .my-element { background: var(--primary-color); color: var(--text-primary); border: 1px solid var(--border-color); }

Key CSS Variables:

  • --primary-color - Main theme color
  • --bg-body - Body background
  • --text-primary - Primary text color
  • --border-color - Border colors
  • --error-color - Error/validation messages
